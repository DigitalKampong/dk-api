## HawkerCentres
Fields needed
- name: string;
- address: string | null;
- regionId: number;



### 1. Create hawker centre



***Endpoint:***

```bash
Method: POST
Type: RAW
URL: {{server_url}}/hawkercentres
```



***Body:***

```js        
{
    "name": "Fake hawker centre",
    "address": "Fake address",
    "regionId": 1
}
```



***More example Requests/Responses:***


##### I. Example Request: Create hawker centre



***Body:***

```js        
{
```
